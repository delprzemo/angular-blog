Material Blog is a modern blog template designed for developers. The main goal was to create a blog that will be:

  • Simple to use - to start using it after 5 mins of introduction
  • Easy to customize - only basic features included. Rest is up to the developer
  • Simple, clean, responsive interface - focus on content, not on extra options
  • Free, open-source - strength is in the community!
  • For IT content - easily highlight code syntax

How to start using it?

Basically, a blog was written in Angular, but you don't need to know it while managing your articles. Before we start adding new articles we need to install it

  1. Firstly, install Angular CLI on your machine. Run this command: npm install -g @angular/cli If you don't have NodeJs then you should install it first

  2. Now clone repository git clone https://github.com/delprzemo/angular-blog.git
  3. In a path where you've cloned repository install dependencies with a command npm install
  4. There is the last command for you - it will run your application ng serve and that's all! Now you can preview a blog by opening http://localhost:4200


Add new article

Articles are kept in HTML. If you are not familiar with HTML then you can use some online editor that will transform content into html. There are a lot of them.
So, in order to add a new article:

  1. Add html file in src/articles/html
  2. Go to src/articles/article.service.ts and add article metadata to returned json. For instance, this is json for this article: return [ { id: 1, title: 'Blog created by developer for developers', shortText: `Simple, customizable, fast, modern, configurable in json, articles in html - finally developers have blog template which is meeting their requirements. Time to look into Material Blog`, date: new Date(2019, 5, 1), html: "easy-blog.html", image: '/assets/blog-item.jpg', imageUrl: '', tags: ['Blog', 'Open source'] }, ... Again, that's it! Just create html file and fill json data. In cloned repo, there are few example articles (including this one).

Customize it

Material Blog is a simple project containing only basic functionalities, so you can customize it according to your needs, add new functionalities or even integrate with another website.

Anyway, it would be nice to change something before publishing it - to make your readers distinguish it from other blogs :) There are few things that you can do:

  1. You can replace top-menu-img.jpg in src/assets with another photo
  2. Change top menu text in src/app/top-menu.component.html
  3. Have fun with styling top-menu (src/app/top-menu.component.less).
    For example change background color: .container-fluid { background-color: rgba(26, 119, 211, 1); }
    or opacity of it: .top-img { opacity: 0.05; }
    Now, it will be slightly different than other blogs :)


Make it public

It's up to you how and where you will deploy your blog. Because it is server-less then you need hosting just for static website. There are a lot of them. Personally, I can recommend netlify.com/ where it is extremely easy to configure CI/CD for a repository, especially when it is placed in github. Now, when I want to add new article it is enough to push it into a repository and netlify will do rest of job:)

It is really worth to share your knowledge and experience with a world. Hope, that blog will make it easier and more fun. Enjoy!
